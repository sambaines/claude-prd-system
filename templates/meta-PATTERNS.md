## ⚠️ Claude: Read This First
This file sits above individual projects and records conventions that
apply across all of them. Before suggesting an approach that has a pattern
recorded here, check whether it contradicts or extends it.
If a project's CLAUDE.md conflicts with an entry here, the project-level
CLAUDE.md takes precedence — but flag the conflict so it can be reconciled.

---

## How to Use This File
- Reference from individual project CLAUDE.md files:
  `Cross-project patterns: ../../meta/PATTERNS.md`
- Add an entry when a decision made in one project should inform all future projects
- Add to LESSONS.md when something caused a real problem and the fix should be permanent
- Patterns here are defaults — individual projects can override with a note in CLAUDE.md
